## Documentation Formats

Documentation is available in two formats:

- Online HTML documentation
- Versioned downloadable PDF documents

## Document Status

Each document is assigned one of the following statuses:

| Status | Meaning |
|---|---|
| Draft | Under development |
| Review | Awaiting review |
| Approved | Approved for use |
| Deprecated | Replaced by a newer document |
| Archived | Retained for historical reference |
